# 📈 Stock Profit Calculator

A simple and responsive Stock Profit Calculator built using HTML, CSS, and JavaScript.

## Features
- Calculate Stock Profit/Loss
- Profit Percentage
- Total Profit
- Simple and Responsive UI
- Mobile Friendly

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Future Updates
- Live NSE/BSE Stock Prices
- Portfolio Tracker
- Watchlist
- Price Alerts
- Market News
- Stock Charts
- Dark Mode

## How to Use
1. Enter Buy Price.
2. Enter Sell Price.
3. Enter Quantity.
4. Click **Calculate**.
5. View your Profit/Loss and Profit Percentage.
